Press A Key to Start

instructions

  1. Press any key to start
  2. The game will start by flashing a color and playing a corresponding sound.
  3. Click on the buttons in the same order as the sequence shown by the game.
  4. If you correctly repeat the sequence, the game will add one more color to the sequence and move to the next level.
  5. Keep repeating the sequence as it gets longer with each level.
  6. If you click a wrong color, the game will display a "Game Over" message.
